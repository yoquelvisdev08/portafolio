Calculator Project

This is a basic calculator application built with JavaScript. The main features include:

  • Perform basic arithmetic operations (addition, subtraction, multiplication, division)
  • Handle decimal numbers
  • Clear and delete functionality
  • Keyboard support for number input and operations
  • Responsive design that works on both desktop and mobile devices

This project showcases my ability to create functional web applications using JavaScript and implement mathematical operations in a user-friendly interface.
